opensoldat website
================

Website for the game [opensoldat](https://github.com/opensoldat/opensoldat)

Requirements
------------
* node 16+

Setup
--------
* cd src
* npm i

Building
--------
* cd src
* npm run build

Generated files will apear in parent directory (e.g. index.html)

Development Server
------------------
* cd src
* npm start
